1. [git branch branch명] 으로 branch 생성
2. [git checkout branch명] 으로 해당 branch로 이동
3. [git add 파일명] -> [git commit]으로 add 뒤 commit까지 하기
4. [git push 원격저장소명 branch명]으로 깃허브에 branch push하기
